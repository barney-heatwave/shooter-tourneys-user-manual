I. Minimum Requirements

Windows

A PC with specs made in 2007 and later is recommended.

    OS: Windows 7 SP1
    Processor: One with 1.5+ GHz
    Storage: 474 MB required for Half-Life; 800+ MB required for Shooter Tourneys.

macOS

    OS: macOS 10.7 Lion
    Storage: 474 MB required for Half-Life; 800+ MB required for Shooter Tourneys.
    For more information, see Steam's documentation on running games on macOS or see Apple's documentation on your iMac/MacBook computer.

Linux

A PC with specs made in 2007 and later is recommended.

    OS: Ubuntu LTS
    Processor: One with 1.5+ GHz
    Storage: 474 MB required for Half-Life; 800+ MB required for Shooter Tourneys.
